1.Calculator

Write a program to make a simple calculator using switch statement that takes operator and two numbers from user and print the result after operation.Note: You have to complete Calculator.Input Format:The input contains a operator symbol(+,-,*,/) along with two natural numbers B and C.

              const Calculator = (A, B, C) =>{  
                switch(A){
                  case "+":
                    return B + C;
                  case "-":
                    return B - C;
                  case "*":
                    return B * C;
                  case "/":
                    return B / C;
                  default:
                  return "Invalid Input";
                }
              };
              Input:+1 2
              Output:3
